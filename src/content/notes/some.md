The key quote from a <cite>Times</cite> "article" on [death by blogging](http://www.nytimes.com/2008/04/06/technology/06sweat.html):

> At the same time, some are starting to wonder if something has gone very rong.

Some wonder how many *some* must be in order to be a reputable number of experts. In the case of this article, some wonder how close that number is to **one**.
